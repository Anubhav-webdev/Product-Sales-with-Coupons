# Product Sales with Coupons

## Overview
This project is a simple web application that allows users to manage a list of products, including adding, editing, and deleting products. Users can also apply coupon codes to receive discounts on their total purchases. The application is built using HTML, CSS, and JavaScript, and it utilizes local storage to persist product data.

## Files
- **index.html**: The main HTML file that contains the structure of the web page, including a table for displaying products, input fields for adding products, and buttons for applying coupons.
- **engine.js**: The JavaScript file that contains the logic for managing products and applying coupons. It includes functions for rendering products, adding or updating products, editing and deleting products, and calculating discounts.
- **style.css**: The CSS file that styles the web page, defining layout, font styles, padding, margins, and table styles to enhance the visual appearance.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the application.
3. Use the input fields to add products and apply coupon codes.

## Usage Guidelines
- Enter a product name and price in the respective input fields and click "Add Product" to add a new product to the list.
- Use the "Edit" button next to a product to modify its details.
- Click the "Delete" button to remove a product from the list.
- Enter a valid coupon code in the "Enter Coupon Code" field and click "Apply" to see the discounted total.

## Coupon Codes
- **SAVE10**: 10% discount
- **FLAT50**: Flat ₹50 off
- **WELCOME**: 20% discount

Enjoy managing your product sales with coupons!
